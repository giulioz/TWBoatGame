# SeaDuel
Assignment for Web Technologies and Applications course.

## Dependencies

- Npm
- Node.js 8.x
- Mongodb
- Android Sdk (for Android build)
- Xcode (for iOS build)

## Building

```bash
npm run install
npm run build-web

# optional
npm run build-android
npm run build-ios
npm run build-desktop
```

## Running

```bash
npm start

# optional
cd mobile-app/SeaDuelMobile
npm run android
npm run ios
cd ..

cd desktop-app
npm start
```
